How to Import a Collection into Postman:
1-Click the file icon on your Postman app.
2- Select import method:
*Importing the file
*Importing the folder
*Importing through link
*Pasting the raw text
3- Choose the file/collection you want to import
4- Hit the open button, and your file/collection will be imported automatically.
